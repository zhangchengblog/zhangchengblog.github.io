laravel框架笔记

1.环境搭建

步骤

  1. 下载composer网址
  2. 检查composer安装完成命令：php artisan –version
  3. 创建一个laravel项目：composer create-project laravel/laravel
  4. 启动laravel：php artisan serve
  5. 服务器配置：选择的时nginx
  6. 在Nginx1.15.11\conf\vhost下的域名.confs文件下添加：try_files $uri $uri/ /index.php?$query_string*;*
  7. 在c盘system32中的hosts文件下添加：127.0.0.1 域名

2.路由

  1. 基本路由
Route::get('/hello1', function () {
    return "Hello word1!";
});
  1. 多种请求方式的路由
Route::any("/any",function(){
    dd("any");
});
//或者
Route::match(["get","post"],"/login",function(){
    dd("match");
});
  1. 定向路由
Route::redirect("/redirect1","/redirect2",301);
  1. 视图路由
Route::view("/wel","welcome");
  1. 必填参数路由
Route::get("/params/{id}",function($id){
    return $id;
});
  1. 可选参数路由
Route::get("/optional/{id?}",function($id="default"){
    return "可选参数：".$id;
});
  1. 正则约束=>路由参数
Route::get("/regular/{id}",function($id){
    return $id;
})->where('id',"[0-9]+");

//或者
Route::get("/regular/{name}",function($name){
    return $name;
})->where('id',"[a-zA-Z]+");

//全局约束  在providers/RouteServiceProvider.php的boot类中
Route::pattern("id","[0-9]+");
  1. 命名
Route::get('givename', function () {
    // 通过路由名称生成 URL
    return route('givename');
})->name('givename');

//含参数
Route::get("/user/{id}/{name}/givename_params",function(){
    return route('givename_params',['id'=>123,"name"=>'lisi']);
})->name("givename_params");

控制器

  1. 基本控制器

创建命令

php artisan make:controller controllername

路由格式

Route::get("url",[controllername::class,"functionname"])
  1. 单行为控制器

创建命令

php artisan make:controller controllername --invakable

路由格式

Route::get("url",controllername::class)
  1. 资源控制器

创建命令

php artisan make:controller controllername --resource

路由格式

Route::get("name",controllername::class)->names(['index'=>"index2"])->parameters(['name'=>'name2'])
