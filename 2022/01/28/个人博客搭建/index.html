#折叠摘要

laravel

个人博客搭建

1.所需工具

  1. node.js
  2. git
  3. hexo
  4. 以上的下载教程网址

2.搭建仓库

步骤

  1. 注册github账号
  2. new一个仓库
  3. 仓库名称写账户名
  4. 全部勾选,完成

3.生成ssh keys

步骤

  1. 随便进入一个文件夹，右键git bash here运行
  2. 输入命令 ssh ，之后会在用户文件夹下生成.ssh文件夹
$ ssh
  1. 进行ssh邮箱绑定,输入之后一直回车。
$ ssh-keygen -t rsa -C "邮箱地址"
  1. 复制.ssh/id_rsa.pub文件下的所有内容
  2. 打开github的settings下的SSH and GPG keys选项
  3. 新建一个ssh keys（名字无所谓，key为之前复制的内容）
  4. 验证ssh是否绑定成功,输入以下命令，输入yes，完成。
$ sh -T git@github.com

4.生成本地博客

  1. 新建一个文件夹，右键git bash here运行
  2. 初始化hexo，（失败就多安装几次）,成功后，会多出很多文件。
$ hexo init
  1. 生成本地博客页面，启动服务器命令：
$ hexo s
  1. 复制网址打开，ctrl+c(关闭服务器)，再次打开会404。

5.博客发布至互联网

  1. 打开上一步文件夹下的_config.yml
  2. 将deploy配置项替换，内容如下：
deploy:
	type: git
	repository: 仓库网址
	branch: main
  1. git bash here运行
  2. 生成页面：
$ hexo g
  1. 验证账号：
$ git config --global user.email "邮箱"
$ git config --global user.name "用户名"
$ hexo d
  1. 接着会弹两个窗，输入用户名，输入密码
  2. 在输入密码之前，打开github settings/Developer settings/Personal access tokens 创建一个令牌
  3. 名字随便取，内容全部勾选
  4. 复制令牌，粘贴至密码输入框
  5. 最后成功将本地文件上传至github仓库
  6. 网址名就是仓库名，打开就能看到自己的博客了！
